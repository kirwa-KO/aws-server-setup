# aws-server-setup

### What setup.sh does:
- install docker
- install docker-compose
- install zsh
- install oh-my-zsh and set zsh as default shell
- add oh myzsh autosuggestions plugin

### Run

```sh
sudo curl -fsSL https://raw.githubusercontent.com/kirwa-KO/aws-server-setup/main/setup.sh | sudo sh
```
